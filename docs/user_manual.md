# Technical User Manual: Metal Powder Handling & Operation

## 1. Material Specifications & Storage
- **Approved Materials**: 316L Stainless Steel, Ti-6Al-4V (Grade 5 Titanium), Inconel 718.
- **Particle Size Distribution (PSD)**: 15–45 µm spherical gas-atomized powder.
- **Storage Protocol**: Store inside hermetically sealed desiccator cabinets filled with Argon gas. Ambient humidity must remain below 15% to prevent powder clumping and moisture-induced print porosity.

## 2. Mandatory Personal Protective Equipment (PPE)
Before touching or opening any powder container, operators must wear:
1. **Respiratory Protection**: NIOSH-approved N100 or P100 particulate respirator mask.
2. **Body Protection**: Anti-static lint-free lab coat or full-body Tyvek suit.
3. **Hand Protection**: Nitrile gloves over conductive anti-static wrist-grounding straps.
4. **Eye Protection**: Impact-resistant safety goggles.

## 3. Pre-Print Machine Preparation Checklist
- [ ] Inspect the recoater blade (`MOT-003`) for nicks, tears, or alignment deviations.
- [ ] Clean the F-Theta scan lens protective window (`OPT-002`) using optical-grade lens wipes and 99.9% Isopropyl Alcohol (IPA).
- [ ] Fill the powder supply hopper to the calculated layer volume height.
- [ ] Close the chamber door and engage the physical safety latch interlock.
- [ ] Open the Argon gas valve (`ENC-002`) and run the oxygen purge sequence until the sensor reads `< 0.1% O2` concentration.
